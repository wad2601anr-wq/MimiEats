# 🌸 MimiEats — AI-Powered Food Ordering Platform

MimiEats is a smart food ordering web application where buyers are in control. Powered by an AI chatbot, an auction-based pricing system, and real-time order tracking — MimiEats makes ordering food smarter, faster, and more fun.


🌐 **Live Buyer:** http://mimieats.infinityfreeapp.com/WAD26/index.html
🏪 **Live Seller:** http://mimieats.infinityfreeapp.com/WAD26/seller.php
---

## ✨ Key Features

- **AI Chatbot (Mimi)** — Describe your craving and Mimi finds the best options automatically
- **Auction System** — All sellers compete for your order, sorted by best price
- **Price Negotiation** — Propose your own price, seller accepts or rejects
- **Live Order Tracking** — Real-time 5-step tracking from kitchen to your door
- **Seller Dashboard** — Full order management, chat, bill builder, and ratings
- **Interactive Map** — OpenStreetMap showing all seller locations
- **Cinematic Video Intro** — Each seller has a unique 15-second animated intro
- **Real-time Chat** — Two-way messaging between buyer and seller via MySQL

---

## 🛠️ Tech Stack

- **Frontend:** HTML5, CSS3, JavaScript
- **Backend:** PHP
- **Database:** MySQL
- **Maps:** Leaflet.js + OpenStreetMap
- **Animations:** HTML5 Canvas API

---

## 🚀 How to Run

1. Download or clone this repository
2. Place the project folder into your `htdocs` directory (XAMPP)
3. Create a database named `mimieats` — tables are auto-created on first run
4. Access `localhost/mimieats/WAD26/index.html` for the Buyer view and `seller.php` for the Seller view
